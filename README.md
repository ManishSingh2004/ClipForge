# ClipForge
Full-stack SaaS short video generator using Next.js, Remotion, and Tailwind. OAuth via Clerk, Neon Postgres (Drizzle ORM), AI scripts (OpenAI), TTS (Google), captions (Assembly AI), images (Replicate). Secure rendering, credits, scalable storage.
